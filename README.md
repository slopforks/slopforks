# slop forks

A leaderboard of popular open-source repos that have been forked and rebuilt with AI.

Built entirely by Claude. No humans were mass-harmed in the making of this website.

## Submit your slop

Open an issue using the [Submit Slop](https://github.com/slopforks/slopforks/issues/new?template=submit-slop.yml) template.

```sh
npm install
npm run dev
```
